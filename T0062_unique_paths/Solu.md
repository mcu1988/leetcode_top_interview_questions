# leetcode t62
- 从矩阵左上角走到右下角的方法数，每次只能向右走或者向下走
- eg
    - m = 3, n = 7，return 28

# 思路1
- 动态规划
- 建立二维表 dp = new int[m][n], dp[i][j]表示（0，0）走到（i，j）的方法数
- 递推关系式 dp[i][j] = dp[i-1][j] + dp[i][j-1]
- 初始时第0行和第0列值是1
- 时间复杂度O(m*n), 空间复杂度O(m*n)

# 思路2
- 把思路1中的dp表压缩到1维
- 方法1中的空间压缩为1行
- 递推关系式 dp[i][j] = dp[i-1][j] + dp[i][j-1]
- dp[i][j]依赖于左边和右边的值，因此可以把dp压缩为1维
    - 可以压缩为1行
    - 可以压缩为1列
- 时间复杂度O(m*n), 空间复杂度O(m)
