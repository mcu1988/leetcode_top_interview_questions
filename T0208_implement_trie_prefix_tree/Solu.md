# leetcode t208
- 实现前缀树
- 功能
    - 插入字符串
    - 搜索字符串
    - 是否有以某个前缀开头的字符串
- 输入字符串值包含小写字母
- eg,
    
    
# 方法
- 建立前缀树，前缀是可以理解为多叉树
- 输入是小写字母，是26叉树
- 节点需要登记以该节点结尾的字符串数量
- 节点结构
    ```java
        class Node {
            Node[] nexts; // 多叉树路径上存储值
            int end; // 以该节点结尾的字符串数量
            Node() {
                // 26个小写字母
                nexts = new Node[26];
                end = 0;
            }
        }
    ```
